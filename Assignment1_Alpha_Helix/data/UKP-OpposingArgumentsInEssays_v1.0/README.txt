Opposing Arguments in Persuasive Essays
---------------------------------------

The corpus contains 402 persuasive essays. The folder contains the following files:

1. Folder „essays“: the 402 essays in plain text

2. folds.tsv: the data splitting for each of the 100 folds of the repeated CV. Each column
   represents a single run and the assignment of each essay in train, test and development    
   set.

3. labels.tsv: the label of each essay (positive or negative).
